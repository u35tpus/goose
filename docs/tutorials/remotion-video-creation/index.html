Create Videos with Remotion Skills

goose can create programmatic videos using Remotion, a framework for creating videos with React. By loading Remotion Agent Skills, goose gains knowledge of best practices for animations, compositions, text effects, and video rendering.

Prerequisitesâ€‹

  • Node.js 18 or later
  • Install Remotion Agent Skills by running npx skills add remotion-dev/skills in your terminal
Remotion License

Remotion is free for individuals and small teams, but requires a commercial license for companies with 3+ employees.

Configurationâ€‹

Enable the Skills extension to allow goose to load and use Agent Skills.

  • goose Desktop
  • goose CLI
  1. Click the button in the top-left to open the sidebar
  2. Click Extensions in the sidebar
  3. Toggle Skills on
  1. Run the configure command:
goose configure
